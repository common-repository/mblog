A chatroom for blog authors

== Description ==

Puts a "wall" into a blog, where blog authors can chat. Wall can be hidden from unregistered users. 
Intended to use in community/corporate blogs, for coordination between authors.

== Installation ==

1. Upload the archive contents to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place widget on page, using 'Widgets' menu in WordPress
4. Specify widget settings (still in 'widgets' menu) and click 'save' button. Plugin is ready to work.
